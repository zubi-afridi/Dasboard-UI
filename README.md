**Analytix** is a high-end, production-ready analytics dashboard designed with **Neumorphism (Soft UI)** principles. It provides a sleek, modern interface for data visualization, project management, and real-time activity tracking, built on the cutting edge of the React ecosystem.

---

## ✨ Features at a Glance

- 🎨 **Neumorphic Design System**: Custom-built soft UI components including cards, buttons, and layouts for a premium tactile feel.
- 📈 **Real-time Analytics**: Interactive charts for user growth, traffic sources, and revenue metrics.
- 🌓 **Adaptive Dark Mode**: Seamlessly transition between light and dark themes with a sophisticated color palette.
- 📱 **Fully Responsive**: Pixel-perfect performance across mobile, tablet, and desktop devices.
- 🛠️ **Modular Architecture**: Clean, reusable component structure using Next.js 15 App Router.
- 🚀 **Performance Focused**: Optimized LCP and zero-unused-CSS driven by Tailwind 4.

## 🚀 Installation & Setup

1. **Clone the project**
   ```bash
   git clone https://github.com/your-username/analytix-dashboard.git
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```
